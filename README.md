# Rock-Paper-Scissors
A simple rock, paper, scissors game made utilising HTML, CSS and JavaScript

## How to play

 - To run the game clone the repository locally and run the code on local dev server or open the `index.html` file in a browser

- To see the result of each round open your developer console by clicking your right mouse button -> Inspect -> Console

## Game features

- The game consists of 5 rounds where you compete against the computer in the classic Rock, Paper, Scissors game
- The game prints the outcome of each round as a message in the developer console
- The game tracks the score of both the player and the computer, and displays who the winner is at the end, or if the game is a tie
- The game will ask the player if they would like to start over once the game is over, or if the player cancels the game before starting
